KDF

KDF stands for "key derivation function". A key derivation function is a function that converts random data into a properly formatted key that can be used by a cryptographic function. The one you probably want to use is PBKDF2, although various other ones are provided as a convenience.

var Bn = require('./bn'); var Privkey = require('./privkey'); var Point = require('./point'); var Pubkey = require('./pubkey'); var Keypair = require('./keypair'); var Hash = require('./hash'); var pbkdf2 = require('pbkdf2-compat'); function KDF() { };
